# 数据结构与算法的python3实现
数据结构与算法的python3版本，有详细注释。项目中包含栈、队列、双端队列、链表等基本数据结构的定义和排序等算法的实现，其中包含一些自己练习的小例子。

目录对应：

1.Stack - 栈

2.Queue - 队列

3.Deque - 双端队列

4.Linked List - 链表

5.Recursive Function - 递归算法

6.Search and Hash - 查找和散列（包含顺序查找、二分查找、散列、基于散列表的映射结构的定义）

7.Sequence - 排序算法（包含冒泡排序、选择排序、插入排序、希尔排序、归并排序、快速排序）

8.Tree - 树
